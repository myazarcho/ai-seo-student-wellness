# AI Prompts Used in This Project

## Keyword Prompt
Generate non-clinical SEO keyword ideas for university students experiencing academic workload and time-management difficulties. Avoid diagnosis, medication advice, therapy claims, and crisis counselling language.

## Search-Intent Prompt
Classify each keyword as informational, navigational, transactional, or support-seeking. Explain the likely user need in one sentence.

## Article Outline Prompt
Create a safe, non-clinical SEO article outline for university students. Use simple English, practical steps, and university support-resource information.

## Safety Review Prompt
Review the article for diagnosis, medical advice, overpromising, blaming language, unsupported claims, and unsafe recommendations. Suggest safer replacements.
