# Step 1 — Project Foundation

## Project Name
AI-Powered Content and SEO Strategy for a Safe Student Mental-Wellness Platform

## Project Type
AI marketing and SEO portfolio project.

## Main Purpose
To create safe, useful, and searchable educational content for university students experiencing everyday academic stress.

## In Scope
- Academic stress education
- Time-management guidance
- Deadline planning
- Exam preparation
- International student adjustment
- University support-resource information

## Out of Scope
- Diagnosis
- Medical advice
- Medication recommendations
- Therapy claims
- Emergency or crisis counselling
