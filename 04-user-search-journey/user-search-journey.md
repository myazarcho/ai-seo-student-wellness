# Step 4 — User Search Journey

## Scenario
Maya has three assignments due next week and does not know which task to start first.

## Search Query
`how to manage multiple assignment deadlines`

## Journey
1. The student feels overwhelmed.
2. The student searches for a practical solution.
3. The student opens an article with a clear title.
4. The article provides a deadline list and prioritization method.
5. The student creates a weekly plan.
6. The article suggests contacting an academic adviser when the workload is not manageable.

## Search Intent
Informational and problem-solving.
