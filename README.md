# AI-Powered Content and SEO Strategy for a Safe Student Mental-Wellness Platform

## Project Overview
This portfolio project demonstrates how AI can support keyword research, search-intent analysis, content planning, safe article drafting, and content evaluation for a non-clinical student mental-wellness platform.

## Target Audience
University students, especially international students, who experience academic workload, multiple deadlines, exam pressure, time-management challenges, and uncertainty about university support services.

## Safety Boundary
This project provides general educational and self-management information. It does not provide diagnosis, medication advice, therapy, or emergency counselling.

## Project Workflow
1. Define the project foundation
2. Create an audience persona
3. Write the problem statement and project goal
4. Map the user search journey
5. Conduct keyword research
6. Build a content strategy
7. Create and evaluate the first SEO article

## Repository Structure
- `01-project-foundation/` — project scope and boundaries
- `02-audience-persona/` — target user profile
- `03-problem-and-goal/` — problem statement and goal
- `04-user-search-journey/` — user journey and search intent
- `05-keyword-research/` — keyword dataset
- `06-content-strategy/` — content pillars and article plan
- `07-first-article/` — first article outline and draft
- `prompts/` — AI prompts used in the workflow
- `assets/` — screenshots and future visuals

## Interactive Dashboard

This project includes a Streamlit dashboard for exploring keyword
research, search intent, content topics, priorities, safety levels,
and the content calendar.

### Run Locally

1. Create the virtual environment:
   `python3 -m venv .venv`

2. Activate it:
   `source .venv/bin/activate`

3. Install dependencies:
   `python -m pip install -r requirements.txt`

4. Run the app:
   `python -m streamlit run 11-dashboard/app.py`
